Página web Scada para planta industrial

Los SCADAs son programas de Software instalable en los equipos, normalmente servidores, que cumplan los requisitos especificados para ello. Un usuario de una planta automatizada podrá ver y controlar desde él todos aquellos equipos integrados en el sistema. Entre los fabricantes de este tipo de software se encuentran Siemens, con su WinCC y el Citect de Schneider o Wonderware.

Teniendo claro el anterior concepto, en este proyecto se fusionan las interfaces web y el sistema de supervisión Scada, pero a diferencia de los software Scada que se encuentran en el mercado, este proyecto se implementó solo con código open source de libre acceso y sin la necesidad de incurrir en altos costos de licencias, por lo tanto este sistema supervisa los principales parámetros del proceso en cada una de las maquinas , además permite descargar desde la misma interfaz un archivo de tipo excel que recolecta toda la información de los parámetros de producción para tener un control real en consumos / unidades producidas.

Este sistema se puede integrar con una gran variedad de dispositivos / PLC autómatas, además el sistema al estar embebido en un entorno web se puede acceder desde cualquier dispositivo móvil, Tablet o computadora sin límite de usuarios. Para mayor comprensión por favor ver el siguiente video:
